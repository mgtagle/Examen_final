# Examen de Métodos Estadísticos en Ciencias Forestales

Este repositorio contiene el material completo para el examen de la unidad de aprendizaje **Métodos Estadísticos en las Ciencias Forestales**.  
Todas las instrucciones, documentos y conjuntos de datos están disponibles directamente en este repositorio para que las y los estudiantes puedan trabajar de manera autónoma.

## Contenido del repositorio

### 1. Documentos (`/docs`)

- `Instrucciones_Examen_Metodos_Estadisticos.docx`  
  Documento con las instrucciones generales del examen: selección del ejercicio, planteamiento de la pregunta de investigación, hipótesis, análisis descriptivo y aplicación del método estadístico.

- `Ejercicios_Metodos_Estadisticos.docx`  
  Contiene los tres ejercicios disponibles para resolver el examen, cada uno con su conjunto de datos correspondiente y la rúbrica de evaluación.

### 2. Datos (`/data`)

Cada archivo contiene 30 observaciones listas para ser analizadas en R, Python, Excel u otro software estadístico.

- `Ejercicio_1.csv`  
  Datos de diámetro a la altura del pecho (**DAP_cm**) y altura total (**Altura_m**) para 30 árboles.  
  Uso sugerido: análisis de **correlación** (Pearson o Spearman).

- `Ejercicio_2.csv`  
  Datos de biomasa aérea (**Biomasa_t**) en dos sitios forestales (columna `Sitio`: A y B).  
  Uso sugerido: **prueba t** para muestras independientes.

- `Ejercicio_3.csv`  
  Datos de densidad arbustiva y cobertura vegetal.  
  Uso sugerido: **correlación lineal**.
